# Smart Pantry Grocery Budget Management System

A web-based pantry and grocery budget management system developed using Next.js, TypeScript, Tailwind CSS, and Supabase.

## Features
- Pantry inventory management
- Grocery budget tracking
- Expiry monitoring
- Waste tracking
- Dashboard analytics
- AI-powered recommendations

## Technologies Used
- Next.js
- TypeScript
- Tailwind CSS
- Supabase
